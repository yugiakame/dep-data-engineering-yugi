# Phase 6 — Packaging & Deployment

**Weeks 21–24 &nbsp;|&nbsp; Milestone: M6 — Live Deployment**

> The final step. A project that lives only on your laptop isn't a portfolio piece. Phase 6 puts your work on the internet, permanently and publicly.

---

## Phase Goal

By the end of Week 24, you will have:

- A dashboard or report artifact with a clear layout tied to your KPIs
- A live project deployed to GitHub Pages with a working public URL
- A polished, professional repo that a stranger can clone and run
- A short presentation or demo script ready to share
- Completed Milestone 6 submission

| Week | Focus | Deliverable |
